Learn more about GrammarParrot

GrammarParrot™ was designed and created using a large array of different technologies. The website was built with HTML, Bootstrap and custom CSS, and made functional using JavaScript. A Flask server supporting the backend of the site is being hosted on Google Cloud platform, and the frontend hosted on Netlify. The site got its domain registered through domain.com. Additionally, two APIs support the website's main functionality, these being AssemblyAI and GrammarBot.

AssemblyAI API

GrammarParrot™ uses the AssemblyAI API to create transcripts of the audio files recorded by the users. These transcripts are displayed to the user for them to review and are passed to the GrammarBot API to scan for corrections.

GrammarBot API

GrammarParrot™ uses the GrammarBot API to check for any grammatical errors taken from the audio transcript.
